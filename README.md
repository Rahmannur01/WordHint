# WordHintEngine (Qt Prototype)

## Description
**WordHintEngine** — это прототип приложения для автоподсказок слов на основе словаря.  

Программа принимает ввод пользователя, ищет похожие слова и предлагает подсказки, используя алгоритм Левенштейна для оценки схожести строк.  

Проект демонстрирует навыки:  
- Создание GUI с помощью **Qt** (QMainWindow, QPushButton, QLabel)  
- Организация кода через **C++ классы**  
- Обработку текстовых файлов и словарей  
- Работа с буфером обмена (**QClipboard**)  
- Глобальный перехват клавиатуры на **Windows**  

Цель — показать **прототип функциональной программы**, которая может быть расширена в полноценное приложение для автоподсказок текста.  

---

## Features
- Загрузка словаря из текстового файла  
- Автоподсказки при вводе текста  
- Выбор подсказки с помощью клавиш Ctrl/Alt  
- Копирование подсказки в буфер обмена  
- Сортировка подсказок по длине  
- Минималистичный GUI с кнопками подсказок  
- Глобальный клавиатурный хук (Windows only)  

---

## How to Run
1. Открой проект в **Qt Creator**  
2. Укажи путь к словарю в `WordHintEngine::fileName`  
3. Скомпилируй и запусти проект  

**⚠ Внимание:** глобальный перехват клавиатуры работает только на Windows.
